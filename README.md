# Spring 핵심 원리 기본 공부

- 현재 이 프로젝트는 스프링을 사용하지 않고, 순수 자바로 구현을 시작하여, 스프링으로까지 이어지는 내용을 담고있습니다.

- 스프링의 원리와 기초 공부를 목표로 합니다.

<br>

## 공부 완료된 내용

- [x] Spring을 사용하지 않고, 순수 java로만 서비스 로직 구현
- [x] AppConfig에 @Configuration과 @Bean으로 의존성 주입
- [x] @ComponentScan과 @Component
- [x] @Autowired
- [x] 스프링 빈의 라이프사이클
